# Right-Clickable Nametags
- a very simple mod for Minecraft that adds the functionality to right click a name tag and name it that way instead of using an Anvil.
- inspired by a post to /r/MinecraftSuggestions by /u/Different-Job-2064 on Reddit
- Requires Fabric Loader for 1.21.11
